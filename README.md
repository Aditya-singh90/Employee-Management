# Employee Management System (Full-Stack)

This is a full-stack web application developed using *Spring Boot* for the backend and *React JS* (Vite) for the frontend.

## 🚀 Features
* *Create:* Add new employees to the system.
* *Read:* View a list of all employees.
* *Update:* Edit existing employee details.
* *Delete:* Remove employees from the database.

## 🛠 Tech Stack
* *Backend:* Java, Spring Boot, Spring Data JPA, MySQL.
* *Frontend:* React JS, Axios, Bootstrap 5.
* *Build Tools:* Maven (Backend), Vite (Frontend).
